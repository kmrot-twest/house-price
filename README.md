# House Price Predictor

A machine learning web application that predicts house prices based on user inputs.

## Features

- Predicts house prices using a K-Nearest Neighbors (KNN) model.
- Flask-based web application.
- Simple HTML user interface.
- Uses a trained scaler and machine learning model.

## Technologies Used

- Python
- Flask
- Scikit-learn
- HTML & CSS
